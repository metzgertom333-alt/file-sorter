# file-sorter 

Simple Python automation tool that organizes files into folders based on their extensions.

## Features

* Sorts files by type (Images, Documents, Videos, Other)
* Automatically creates folders
* Handles duplicate filenames (`_1`, `_2`, ...)
* Works on Windows

## How to use

1. Put files into the target folder.
2. Update the folder path in the script if needed.
3. Run:

```bash
python file_sorter.py
```

## Requirements

* Python 3.9+
